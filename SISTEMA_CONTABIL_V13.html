📋 Como configurar — Passo a Passo
1. Crie um projeto gratuito no Supabase
Acesse supabase.com → New Project → anote a URL e a anon/public key.

2. Execute o SQL abaixo no Editor SQL do Supabase
Vá em SQL Editor → New Query, cole o script e clique em Run.

-- ContaPro V13 — Script de criação das tabelas no Supabase -- Cole este SQL no SQL Editor do seu projeto Supabase e clique em Run create table if not exists empresas ( id bigint generated always as identity primary key, razao text not null, cnpj text, tipo text default 'Ltda', regime text default 'Simples Nacional', exercicio text default '2025', status text default 'ativo', created_at timestamptz default now() ); create table if not exists plano_contas ( id bigint generated always as identity primary key, empresa_id bigint references empresas(id) on delete cascade, cod text not null, nome text not null, tipo text, nat text, nivel integer ); create table if not exists lancamentos ( id bigint primary key, empresa_id bigint references empresas(id) on delete cascade, data date not null, doc text, deb text, cred text, hist text, valor numeric(15,2), tipo text default 'normal', status text default 'ativo', conciliado boolean default false, origem_ofx boolean default false, origem_ap boolean default false, origem_ap_id bigint, origem_ar boolean default false, origem_ar_id bigint ); create table if not exists ofx_data ( id bigint generated always as identity primary key, empresa_id bigint references empresas(id) on delete cascade, data date, descricao text, valor numeric(15,2), status text default 'pendente', lancado_id text, conta_bancaria_id bigint ); create table if not exists contas_bancarias ( id bigint generated always as identity primary key, empresa_id bigint references empresas(id) on delete cascade, banco text, tipo text, agencia text, numero text, conta_contabil text, saldo_inicial numeric(15,2) default 0, status text default 'ativa', obs text ); create table if not exists saldos_abertura ( id bigint generated always as identity primary key, empresa_id bigint references empresas(id) on delete cascade, cod text, saldo_d numeric(15,2) default 0, saldo_c numeric(15,2) default 0 ); create table if not exists contas_pagar ( id bigint primary key, empresa_id bigint references empresas(id) on delete cascade, fornecedor text, descricao text, doc text, deb text, cred text, emissao date, venc date, valor numeric(15,2), status text default 'aberto', pago_em date, banco_cc bigint ); create table if not exists contas_receber ( id bigint primary key, empresa_id bigint references empresas(id) on delete cascade, cliente text, descricao text, doc text, deb text, cred text, emissao date, venc date, valor numeric(15,2), status text default 'aberto', recebido_em date, banco_cc bigint ); -- Liberar acesso público (RLS desativado para simplificar) alter table empresas disable row level security; alter table plano_contas disable row level security; alter table lancamentos disable row level security; alter table ofx_data disable row level security; alter table contas_bancarias disable row level security; alter table saldos_abertura disable row level security; alter table contas_pagar disable row level security; alter table contas_receber disable row level security;


3. Cole a URL e a chave abaixo e clique em Salvar e Conectar
